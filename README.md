# Projeto Dao JDBC

# Sobre o projeto

O projeto foi feito para fazer CRUD afim de aprimorar os conhecimentos sobre a camada DAO, JDBC e a utilização de banco de dados MySql em uma aplicação.

# Tecnologias utilizadas
## Back end
- Java
- JDBC
## Banco de Dados
- MySql Workbench
